# Udaya Kumar Sivagurunathan — Portfolio

Cinematic portfolio of Udaya Kumar Sivagurunathan: Lead UX Designer & Agentic AI Architect.

**Live:** https://uday-portfolio-site-five.vercel.app

## Stack

- Next.js 14 (App Router) + TypeScript
- Framer Motion (scroll-synced hero, card deck, scroll-scrubbed timeline)
- Tailwind CSS, Poppins, `#acec00` accent

## Develop

```bash
npm install
npm run dev   # http://localhost:3000
```

Pushes to `main` auto-deploy via Vercel.
